### Generate an example

```bash title="Create template controller file" {2}

```

## Troubleshooting

- **ImportError on your method path**  
  Double-check the dotted `method="package.module:function"` matches your agent code and the project root is on `PYTHONPATH`.

- **401 / missing secrets**  
  Ensure required `Parameter` items marked `is_environment=True` exist in your shell or `.env`.

- **Port already in use**  
  Change the port: `sv_server.launch(host="0.0.0.0", port=8010)`.

- **The UI doesnâ€™t show fields**  
  Confirm each field dict includes `name`, `type`, `field_type`, and `required`.
