## <i className="bi bi-3-circle"></i> Scaffold the controller {#3}

### Generate an example

Generate a starter controller in your project:

```bash title="Create template controller file" {2}
supervaizer scaffold
> Success: Created an example file at supervaizer_control_example.py
```

This creates **`supervaizer_control_example.py`**. You‚Äôll customize it to:

- Define **agent parameters** (secrets, env, required inputs)
- Define **agent methods** (start/stop/status, etc.)
- Map those methods to **your agent‚Äôs functions**

### Shortcut

Get the fully customized controller from GitHub [Email-AI-Agent Supervaizer fork](https://github.com/alain-sv/Email-AI-Agent/blob/supervaizer/supervaizer_email_agent_controller.py).

```js reference title="Full example controller supervaizer_email_agent_controller.py"
https://github.com/alain-sv/Email-AI-Agent/blob/supervaizer/supervaizer_email_agent_controller.py#L1-L15
```

For detailed instructions on customizing your controller, see the **[Controller Setup Guide](setup)**.

## <i className="bi bi-4-circle"></i> Configure your supervaize account & environment {#4}

### Supervaize account and credentials

[Create your developer account](/docs/fleet/setup-development-environment#getting_started) on the Supervaize platform.

Create your API Key and collect your environment variables.

:::tip
Use [direnv](https://direnv.net/) for easy management of environment variables in `.envrc` file.

Or setup [python-dotenv](https://github.com/theskumar/python-dotenv) in your python project.
:::

## Troubleshooting

- **supervaizer start** stops after launch

  Debug using direct python start.

```bash
uv run python supervaizer_control.py
```

- **Error sending event SERVER_REGISTER**

```
ERROR | ‚ùå Error sending event SERVER_REGISTER : Client error '403 Forbidden' for url 'https://stage.supervaize.com/w/team_1/api/v1/ctrl-events/'
```

Check at https://stage.supervaize.com/users/developer/ that the API key you are using is active.

- **Port already in use**  
  Change the port: `sv_server.launch(host="0.0.0.0", port=8010)`.
