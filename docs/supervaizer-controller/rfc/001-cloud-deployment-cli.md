# RFC-001: Cloud Deployment CLI

Specification for the Supervaizer Cloud Deployment CLI.

## Summary

The Cloud Deployment CLI automates deployment of Supervaizer agents to GCP Cloud Run, AWS App Runner, and DigitalOcean App Platform, with plan/up/down/status commands, health verification, and rollback support.

## Documentation

See the [Cloud Deployment Guide](../deploy) for the full user-facing documentation and usage.
